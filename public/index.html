Documentação da API

Integre seu workflow n8n com o catálogo de tutoriais usando o webhook abaixo

1. Configure o Trigger no n8n

Use o node 'Google Drive Trigger' para monitorar a pasta de tutoriais

2. Processe com IA

Use Whisper para transcrição e GPT-4 para gerar resumo e tags

3. Envie para o Webhook

Configure o HTTP Request para enviar o payload formatado

4. Visualize no Dashboard

O vídeo aparecerá automaticamente no catálogo

Endpoint do Webhook
POST /api/webhook/video

* No Base44, use a integração de backend functions para criar este endpoint
